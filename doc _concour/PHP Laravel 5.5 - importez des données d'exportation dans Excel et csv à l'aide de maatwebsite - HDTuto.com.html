PHP Laravel 5.5 - Importer des données d'exportation dans Excel et CSV à l'aide de maatwebsite


Salut les gars,

Dans cet article, je vous ferai savoir comment importer et exporter un fichier Excel ou csv dans votre application Laravel 5.5. Ici, je vais vous montrer un exemple complet, étape par étape, pour importer un fichier Excel dans une base de données et l’exporter à partir de la base de données, puis le télécharger.

Nous utiliserons le package maatwebsite composer pour importer des données d’exportation dans Excel et dans un fichier csv. Le package maatwebsite est populaire pour générer un fichier csv et un fichier excel. Ils fournissent également à importer le fichier csv à la base de données.

Ici, dans cet article, je vais vous donner un exemple simple et petit d'importation de données d'exportation dans Excel et CSV à l'aide du package maatwebsite composer. Il suffit donc de suivre quelques étapes pour faire ceci:

Étape 1: Installez le package maatwebsite Composer

La première chose à faire est d’installer le paquetage Laravel maatwebsite pour utiliser la façade Excel. Il suffit donc d’exécuter la commande bellow composer pour installer le paquet.

compositeur besoin de maatwebsite / excel

    ...

Étape 2: Créer une table de produits et un modèle

Ensuite, nous devons créer une table de migration pour les produits à l' aide de la commande artisanale Laravel 5.5 php, donc première commande ci-dessous:

Fabrication artisanale php: migration create_products_table

Après la commande ci-dessus, nous trouverons un fichier dans la base de données de chemin / migrations suivante et vous devrez mettre le code ci-dessous dans votre fichier de migration pour créer la table des produits.

Étape 3: Ajouter des itinéraires

Dans la troisième étape, nous devons créer de nouvelles routes pour les données d'exportation et d'importation. alors ouvrez votre fichier routes / web.php et ajoutez la route suivante.

routes / web.php

Route :: get ('import-export-view', 'ExcelController @ importExportView') -> nom ('import.export.view');
Route :: post ('import-file', 'ExcelController @ importFile') -> name ('import.file');
Route :: get ('export-file / {type}', 'ExcelController @ exportFile') -> nom ('export.file');
