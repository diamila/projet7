Export fichier excel Laravel | Maatwebsite/excel Version 3.1

Posted on 20 décembre 201820 décembre 2018 by Marty

Bonjour, aujourd’hui je vous présente comment exporter un fichier excel depuis Laravel avec Maatwebsite/excel.

étape 1 ->
tapez la commande suivante :
composer require maatwebsite/excel

étape 2 ->
Allez dans le fichier « config/app.php » et rajouter dans le ServiceProviders:
Maatwebsite\Excel\ExcelServiceProvider::class,
Et dans les Aliases:
'Excel' => Maatwebsite\Excel\Facades\Excel::class,
puis tapez la commande:
php artisan vendor:publish

étape 3 ->
tapez la commande suivante (Vous pouvez choisir un autre nom):
php artisan make:export UsersExport --model=User
Vérifier bien que votre model existe dans le dossier « app »
A la suite de la commande un dossier et un nouveau fichier va apparaitre dans « app/Exports ».
A l’intérieure du fichier « UsersExport » vous pouvez modifier la function « collection » par le nom de la base que vous souhaitez récupérer.

étape 4 ->
créer votre controller avec la commande suivante:
php artisan make:controller UserController (ou par le nom que vous souhaitez).
A l’intérieure placer le code suivant :


use App\Exports\UsersExport;
use Maatwebsite\Excel\Facades\Excel;
use App\Http\Controllers\Controller;

class UsersController extends Controller 
{
    public function export() 
    {
        return Excel::download(new UsersExport, 'users.xlsx');
        // Vous devez remplacer le nom par rapport au votre
    }
}

étape 5 ->
Maintenant nous allons créer la View (« resources/views »), pour moi j’ai fait un simple tableau avec  Bootstrap relier à ma base de donnés avec le nom « export_excel.blade.php ». 

<!DOCTYPE html>
<html lang="en" dir="ltr">
  <head>
    <meta charset="utf-8">
    <title>test</title>
    <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css" integrity="sha384-MCw98/SFnGE8fJT3GXwEOngsV7Zt27NXFoaoApmYm81iuXoPkFOJwJ8ERdknLPMO" crossorigin="anonymous">
  </head>
  <body>
<a href="{{ route('export_excel.excel') }}" >Test export</a>
    <table class="table container">
  <thead>
    <tr>
      <th scope="col">ip</th>
      <th scope="col">qid</th>
    </tr>
  </thead>
  <tbody>

    @foreach($customer_data as $customer)
    <tr>
      <td>{{ $customer -> ip}}</td>
      <td>{{ $customer -> qid}}</td>
    </tr>
    @endforeach
  </tbody>
</table>
  </body>
</html>

étape 6 ->
Nous pouvons maintenant créer notre route (« routes/web.php »)

Route::get(‘excel_export’, ‘ExportExcelController@Export’);
Route::get(‘/excel_export/excel’, ‘ExportExcelController@excel’)->name(‘export_excel.excel’);

Maintenant en cliquant sur le lien de votre page vous pouvez télécharger votre tableau au format xls.
